# Unit Converter MVP

A simple MVP unit converter with focused functionality:

- Distance: meters, kilometers, feet
- Weight: grams, kilograms, pounds
- Volume: liters, gallons, milliliters
- Dynamic target-unit filtering based on selected source unit

## Run

Open `index.html` in a browser.

## Test

```bash
npm test
```

## Foundation for future expansion

- Add a category selector UI to support larger unit sets.
- Persist recent conversions in local storage.
- Add conversion history and favorites.
- Add locale-aware number input/output preferences.
- Add API-backed unit metadata for enterprise-scale coverage.
